Setup and Run Instructions

WARNING: No Current Windows Support
1. System Environment Setup
For macOS:
brew install curl blast

For Linux (Ubuntu/Debian):
sudo apt-get update && sudo apt-get install curl ncbi-blast+

Install Python dependencies:
pip3 install pandas numpy scikit-learn goatools

2. Data Acquisition
Download the Gene Ontology graph file:
curl -o go-basic.obo http://current.geneontology.org/ontology/go-basic.obo

Download the Swiss-Prot database stream (~500MB):
sh get_data.sh

3. Process and Convert Data
Filter and split the raw data:
python3 make_data.py

Convert CSV files to FASTA format:
python3 dftofasfa.py

4. Run Local BLAST Alignments
cd blast

Run the script for your target organism:
sh blast_methanocaldococcus_jannaschii.sh    # Archaea (~5 mins)
sh blast_escherichia_coli.sh                 # E. coli (~15-30 mins)
sh blast_mycobacterium_tuberculosis.sh       # M. Tuberculosis (~15-30 mins)
sh blast_saccharomyces_cerevisiae.sh         # Yeast (~30-45 mins)
sh blast_homo_sapiens.sh                     # Human (45+ mins)

Return to the project root directory:
cd ..

5. Evaluate Model and Extract TDA Data
Note: Open code_for_models/NN.py and change the "name" variable on line 8 to your target organism (e.g., name = "homo_sapiens") before running.

python3 code_for_models/NN.py
