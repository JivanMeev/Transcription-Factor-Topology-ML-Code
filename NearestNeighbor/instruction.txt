Setup and Run Instructions

WARNING: No Current Windows Support
1. System Environment Setup
For macOS:
brew install curl blast

For Linux (Ubuntu/Debian):
sudo apt-get update && sudo apt-get install curl ncbi-blast+

Install Python dependencies:
pip3 install pandas numpy scikit-learn goatools

2. Data Acquisition

Make Directories:
mkdir data
mkdir blast

Download the Swiss-Prot database stream (~500MB):
sh get_data.sh

3. Process and Convert Data
Filter and split the raw data:
python3 make_data.py

Convert CSV files to FASTA format:
python3 dftofasfa.py

4. Run Local BLAST Alignments
cd blast

Run the global test script
sh run_all_downsample_blast

Return to the project root directory:
cd ..

5. Evaluate Model and Extract TDA Data
python3 NN.py
